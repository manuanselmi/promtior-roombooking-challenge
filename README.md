# RoomBooking — Promtior Technical Challenge

Chatbot with tool-calling capabilities that lets users book meeting rooms at the
Cubo Itaú office through a conversational interface.

Built with FastAPI, LangChain (`create_agent` / LangGraph) and SQLite.
